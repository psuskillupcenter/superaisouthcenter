Learning Journey

From AI Awareness to Working Prototype

เส้นทางการเรียนรู้ถูกออกแบบให้เริ่มจากการคิดอย่างรู้เท่าทัน ก่อนเข้าสู่การใช้ Prompt การจัดการความเสี่ยงข้อมูล และการสร้างผลงานต้นแบบ

01

AI Literacy

เข้าใจศักยภาพ ข้อจำกัด ความเสี่ยง และบทบาทของ AI ต่อการทำงานยุคใหม่

02

Critical Prompting

ฝึกตั้งคำถาม ออกแบบ Prompt ตรวจสอบคำตอบ และพัฒนาผลลัพธ์ให้เหมาะกับบริบทงาน

03

Safe AI Use

เรียนรู้การใช้ข้อมูลอย่างปลอดภัย ระวังข้อมูลส่วนบุคคล ข้อมูลองค์กร และความน่าเชื่อถือของผลลัพธ์

04

Vibe Coding

เปลี่ยนปัญหางานให้เป็น Prototype ระบบขนาดเล็กหรือ Dashboard ที่ทดลองใช้งานได้จริง

Why Join

Learn AI. Build Smarter. Create Impact.

เข้าร่วมเพื่อยกระดับทักษะ AI ทั้งด้านการคิด การใช้งาน การสร้างต้นแบบ และการนำเสนอผลงานอย่างมืออาชีพ

🧠

Critical AI Mindset

ฝึกคิดเชิงวิพากษ์ ไม่เชื่อ AI ทันที แต่รู้วิธีตั้งคำถาม ตรวจสอบ และปรับปรุงผลลัพธ์

🔐

Safe Prompting

รู้เท่าทันความเสี่ยงด้านข้อมูล ความเป็นส่วนตัว ความถูกต้อง และจริยธรรมของการใช้ AI

⚙️

Prototype Skills

ทดลองสร้างระบบหรือ Dashboard ขนาดเล็กด้วยแนวคิด Vibe Coding เพื่อแก้โจทย์งานจริง

🚀

Portfolio Ready

ได้ผลงานต้นแบบและประสบการณ์นำเสนอ ที่สามารถต่อยอดเป็น Portfolio หรือโปรเจกต์จริงได้
